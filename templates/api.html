API:background-image

API 简介：获取随机背景图片

端点：/api/background-image

请求类型: GET

参数

  • 无需传入任何参数。

请求

GET /api/background-image

{
	GET /api/background-image HTTP/1.1
	Host: your-api-domain
}

请将 your-api-domain 替换为实际部署该 API 的域名或 IP 地址。

成功响应

{
	HTTP/1.1 200 OK
	Content-Type: [image content type]
	Content-Length: [size in bytes]

	[image data]

}

失败响应

{
	HTTP/1.1 404 Not Found
	Content-Type: text/plain
	No images available
}

备注

这个 API 允许客户端获取一个随机的背景图片。客户端通过向 /api/background-image 发送 GET 请求来获取图片数据。如果有可用的图片，将返回 HTTP 状态码 200 和相应的图片数据；如果没有可用的图片，则返回 HTTP 状态码 404 和消息 "No images available"。

API 简介：添加用户

端点：POST /add_user

权限要求

  • 需要管理员权限。
  • 会话中的 IP 地址将被检查以确保安全性。

请求参数

参数 类型 描述
username 字符串 新用户的用户名。
password 字符串 新用户的密码。

成功响应

HTTP/1.1 200 OK
Content-Type: application/json

{
    "message": "用户添加成功。"
}

错误响应

HTTP/1.1 400 Bad Request
Content-Type: application/json

{
    "error": "用户名和密码不能为空。"
}
            
HTTP/1.1 400 Bad Request
Content-Type: application/json

{
    "error": "用户名已存在。"
}

备注

该接口用于管理员添加新用户到系统中。如果用户名或密码未提供或者用户名已存在，将返回相应的错误消息。
